### `> how.i.operate`

```yaml
principles:
  - A decade of scars taught me when to trust the model — and when not to.
  - Natural language is the new compile target. Types and tests are still law.
  - Automate the boring 80%. Obsess over the critical 20%.
  - Ship small. Observe everything. Roll back without fear.

workflow:
  - spec     : 3-line intent → agent scaffolds the module
  - tests    : red first, let the model converge to green
  - review   : diff like a senior human — reject politely, often
  - harden   : types, property tests, observability, rollback
  - deploy   : canaries, feature flags, blameless retros

non-negotiables:
  - Reproducible builds · Observable systems · Secure-by-default.
  - OWASP Top 10 — certified in Rust & Angular.
  - Correctness > cleverness · Simple > smart.
```
